# MediHealth Appointments

This is a small web application for booking MediHealth appointments.  
It was created for T Level Digital Production, Design and Development – Task 2 (Week 1).

## What this project does

- Lets a user enter a patient name, appointment date and notes in a form.
- Sends the data to a Supabase database table called `appointments`.
- Shows a list of all appointments on the page.
